git教程总结
##创建版本库
###首先配置用户
		git config --global user.name "Your Name"
		git config --global user.email "email@xx.com"
	1.新建目录，
		mkdir git
		cd git
	2.通过git init初始化
		git init

##添加文件
1.添加文件
	git add readme.txt
2.添加到仓库
	git commit -m "message"
git 可以分别add文件，然后用 commit一次性提交

D.时刻查看文件的改动   	
	git status
