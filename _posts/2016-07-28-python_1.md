# 中文编码

如果在源代码中插入中文的话，可能会报编码错误，需要在头文件中加入：

# -*- coding: UTF-8 -*-

# 字符表示

```bash 
print '\\\n\\\'

## 打印 
# \
# \

## 防止转义

print r'\\\n\\\'

## 多行内容使用三引号 ‘'''...'''
print ''' hello
world
yubo
'''
## 打印布尔值
print 3>2

## unicode编码转换成utf-8
print u'ABC'.encode('utf-8')
print u'中文'.encode('utf-8')
```
