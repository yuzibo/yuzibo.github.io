我认为是有多少个26,就是多少个A，结果人家的答案是"ALM",我的思路完全是错误的。

先把我的错误代码放上来，好好看看自己的弱点

```c

char str[100];
char *convertToTitle(int n){
	int len = n/26 + 1;
	int i;
	for (i = 0; i < len - 1 ;i++){
		str[i] = 'A';
	}
	int last_num = n - n/26*26 -1;
	str[i] = (char)(last_num + 'A');
	str[len] = '\0';
	return str;

}
```
