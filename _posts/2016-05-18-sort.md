## 2.2 bubble sort
所谓的冒泡排序大家都听说过，这里直接上代码，不过你要注意那个标志符的作用。我
还是在老地方犯错：先是一个while、接着for，之间填一个sorted=1;这样效率上会有
提升。

```c
/*我的困难局限于标志变量的位置*/
void bubblesort(int a[],int n){
	int i,j,temp,sorted;
	i = sorted = 0;
	while( (i < n - 1) && (!sorted)){
		sorted = 1;
		for(j = n - 1; j > i; j--){
			if(a[j] < a[j-1]){
				temp = a[j];
				a[j] = a[j-1];
				a[j-1] = temp;
				sorted = 0;
			}
		}
	}
}
```
