# 在命令行下新建窗口

## dos
新建窗口: **start .**打开cmd所在的窗口,或者说open windows Explorer from cmd prompt here.

相反方向的话直接在地址栏中输入**cmd**.即可打开该窗口对应的cmd窗口。

```bash
copy filename1 filename2
```

## linux
以下桌面系统是在默认状态下.
Gnome(Ubuntu): 使用命令：

	       nautil .

Xfce(debian):

	xdg-open .
